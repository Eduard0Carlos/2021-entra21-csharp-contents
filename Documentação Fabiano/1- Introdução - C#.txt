		---Introdução---

Bibliotecas: São pacotes que extendem as funções de certa línguagem de programação;

Plataforma: É um conjunto de bibliotecas. .NET (dot net) é um exemplo de plataforma (ou ambiente de execução);

C# é uma linguagem de programação que usa as bibliotecas da plataforma .NET;

OBS: Na linguagem C#, É OBRIGATÓRIO o uso de ponto & vírgula (;) no final de "quase" todas as linhas do código.

.NET FRAMEWORK: (Usada apenas em Windows)
.NET CORE: (Usada em Linux, Mac, e Windows)
XAMARIN: (Usada em IOS, Android, e MAC)

[CURIOSIDADE] Ao criar um novo projeto, o sistema automaticamente define os modulos (ou pacotes) que você vai usar durante seu programa (de acordo com o que você especificou anteriormente). Isso economiza muito memória e deixa o programa mais leve.
Por exemplo ao criar um projeto no qual você especificou que vai usar os componentes do Windows Form, o programa automaticamente especifica que você vai usar os componentes do Windows Form 
-using System;
-using System.WindowsForms;
